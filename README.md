<h1 align="center">PakePlus-Android</h1>  
<p align="center">比亚迪浏览器（byd browser），这并不是一个真正的浏览器，这个程序用于拉起车机自带的浏览器。</p>
<p>核心是scripts/www/index.html，原理是使用html跳转一下链接，就会唤起默认浏览器。在车机文件管理器执行这个html也能实现同样功能。</p>
